# Hub de filmes

## Endpoints

### em relação a usuário

- GET /users
- GET /users/:userId
- POST /users
- PUT /users/:userId
- DELETE /users/:userId

### em relação ao filme

- GET /films
- GET /films/:filmId
- PUT /films/:filmId
- POST /films
- DELETE /films/:filmId


### em relação a categoria

- GET /categories
- GET /categories/:categoryId
- PUT /categories/:categoryId
- POST /categories
- DELETE /categories
